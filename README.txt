This project is created for first year's project in college in computer programming class
.
Before you can run the program !
1. Users must have installed 'PYTHON' on the computer
2. Users must have installed 'PIP' on the computer
3. Users must have installed 'PANDAS' on the computer
4. This program has 3 main function 
4.1 Enter item : In this function users must use this format 'date +income/-expense description '
4.2 Display : In this function user must fill the program with date format EX. 2/2/2002 ....the program will show data about the date
4.3 Load : In this function user must fill the program with month format EX. 2/2002 ....the program will show data about the month 
4.3.1 The Load function will ask users if user want to show all list in that month. if not the program will comeback to menu autometically
5. Exit : if user select this the program will stop running
